* *Сборка:* `DocsVision.Edi.ObjectModel.dll`

----

== Свойства

[cols=",",options="header",]
|===
|Имя |Описание

|`Comment` |Задает или возвращает содержимое комментария.
|`SignatureId` |Задает или возвращает идентификатор подписи, которой подписан комментарий.
|===
