= Исходящий УКД

Карточки вида _Исходящий УКД_ предназначены для корректировки сумм и количества товаров в переданном УПД. Пример карточки приведен на следующем рисунке.

image::outgoingUKD.png[Пример исходящего УКД]

Список элементов карточки _Исходящий УКД_ аналогичен списку элементов карточки xref:CardOutgoingUPD.adoc[Исходящий УПД]. *Функция УПД* может принимать значения: "КСЧФ", "ДИС" и "КСЧФДИС".
